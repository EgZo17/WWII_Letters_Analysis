#  WWII Letters Analysis

**NLP-проект для анализа тональности и эмоций в письмах периода ВОВ (1941–1945)**

Этот проект представляет собой автоматизированный пайплайн для сбора, обработки и анализа текстов фронтовых писем. С помощью нейросетевой модели (RuBERT) и лингвистического анализа определяются динамика настроения, средняя длина текстов и доминирующие эмоции солдат на протяжении всех лет войны.

## Как запустить

### 1) Установка зависимостей

Убедитесь, что у вас установлен Python 3.8+. Создайте виртуальное окружение и установите библиотеки.
Для этого запустите в терминале в корневой папке проекта команды:

```text
python -m venv .venv
.venv\Scripts\activate  # Для Windows
pip install -r requirements.txt
```

### 2) Сбор данных (Опционально)

Скрипт помогает вручную добавлять письма в базу. Запускается интерактивно, позволяет вставлять тексты с сохранением абзацев:

```text
python scripts/collect_letters.py
```

### 3) Обработка данных (NLP)

Этот шаг прогоняет тексты через модель RuBERT, определяет тональность (позитив/негатив/нейтрал) и эмоции по ключевым словам (страх, надежда, тоска и т.д.). Результат сохраняется в ```data/letters_enriched.json```.

```text
python scripts/process_letters.py
```

### 4) Анализ и визуализация

Запускает полный пайплайн: проверка статистических гипотез, построение графиков динамики и генерация итогового отчёта.

```text
python src/main.py
```

Результаты (графики и JSON-отчёт) сохраняются в папку ```results/```.

## Описание исследования

Проект проверяет две исторические гипотезы на основе данных:

1) **Длина писем:** Становятся ли письма короче к концу войны?
2) **Тональность:** Уменьшается ли доля негативных эмоций к победе?

Используется комбинированный подход: классификация тональности нейросетью + словарный метод для выделения специфических эмоций.
